Mangalam HDPE Pipes & Coils Web Showcase
A high-performance, responsive product landing page for Mangalam HDPE Pipes. This project showcases premium High-Density Polyethylene pipes through a modern interface featuring interactive elements like an image zoom gallery, technical specifications, and a dynamic sticky header.

🚀 Features
Interactive Product Showcase:

Custom image carousel with thumbnail navigation.

Advanced Image Zoom: A lens-based magnification system for viewing technical details of the pipes.

Conversion-Focused UI:

Dynamic Sticky Header: Appears on scroll to show pricing and a "Get Custom Quote" call-to-action (CTA).

Certification badges (BIS, ISO, CE) prominently displayed for trust.

Technical Information:

Clean, structured technical specification table.

Feature grid highlighting chemical resistance, longevity, and sustainability.

Responsive Design: Optimized for desktop, tablet, and mobile devices using a mobile-first approach.

SEO Optimized: Semantic HTML5 with meaningful meta descriptions and accessibility attributes.

🛠️ Tech Stack
HTML5: Semantic structure.

CSS3: Custom properties (variables), Flexbox, and CSS Grid for layout.

JavaScript (Vanilla): For carousel logic, sticky header toggles, and image zoom functionality.

Google Fonts: "Plus Jakarta Sans" for a modern, professional look.

⚙️ Key Components
Sticky Header
The #stickyHeader element provides a seamless transition for users. It contains the product title and a price range, ensuring the primary conversion goal is always visible.

Technical Table
The .specs__table handles complex data ranges (e.g., 20mm to 1600mm) while remaining readable on smaller screens via a responsive wrapper.

Product Certifications
Utilizes a "pill" design pattern to quickly communicate compliance with industry standards like IS 5984 and ISO 4427.
